<h1 align="center"><b>Quad9DNS4Magisk</b></h1>
<h4 align="center">This Module makes your Device to use Quad9 DNS Servers</h4>
<br />

## Quad9 DNS Servers:
* 9.9.9.9
* 149.112.112.112

## Caution:
It is not guaranteed to work on Cellular Networks on all Devices
<br />

## Requirements:
Magisk installed (v17+)
<br />
